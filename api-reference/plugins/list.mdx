# List Plugins

Returns a list of all registered verification plugins, including built-in and custom plugins.

## Request

```bash
curl https://agentgate.mynewapi.com/api/v1/verification/plugins \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array" required>
  Array of plugin configurations

  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique plugin identifier
    </ResponseField>
    <ResponseField name="name" type="string">
      Human-readable plugin name
    </ResponseField>
    <ResponseField name="description" type="string">
      Plugin description
    </ResponseField>
    <ResponseField name="level" type="string">
      Verification level: `L0`, `L1`, `L2`, `L3`, or `custom`
    </ResponseField>
    <ResponseField name="priority" type="number">
      Execution priority (lower runs first)
    </ResponseField>
    <ResponseField name="enabled" type="boolean">
      Whether plugin is enabled
    </ResponseField>
    <ResponseField name="continueOnFail" type="boolean">
      Whether to continue verification if this plugin fails
    </ResponseField>
    <ResponseField name="options" type="object">
      Plugin-specific options
    </ResponseField>
  </Expandable>
</ResponseField>
